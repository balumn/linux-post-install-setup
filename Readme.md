# Linux Post Installation setup

## Post Ubuntu18.04 Installation
#### Step 1

Run the bash script

    sudo bash ubuntu18_04.sh

#### Step 2
 In Gnome-tweaks install gnome extensions

	    NetSpeed
	    Caffiene
	    Clipboard Indicator
	    GSConnect
	    Time++
	    User Themes

#### Step 3
 Configure shell
 Some of basic fish, bash configs are added. Replace'em onto respective places.
